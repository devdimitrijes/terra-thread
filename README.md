🛍️ Handmade Products Website
A modern, responsive website designed for a handmade products brand, focused on clean UI/UX, smooth interactions, and an elegant visual experience.

✨ Overview
This project is a minimal and aesthetic website that showcases handmade products in a visually appealing and user-friendly way. The design follows a “vibe coding” approach — clean, modern, and interactive without unnecessary complexity.

📄 Pages

🏠 Home Page
Hero section with a strong headline and call-to-action button
Short introduction about the brand
Featured products preview (3–6 items)
Smooth scrolling and subtle animations

🖼️ Gallery / Products Page
Grid layout for displaying products
Images with short descriptions
Clean spacing and modern layout
Hover effects for interactivity

📖 About Page
Story and background of the brand
Clean typography for readability
Simple and elegant layout

📬 Contact Page
Contact form (name, email, message)
Social media links
Minimal and clean design

🎨 Design Features
Minimal and modern UI
Soft, neutral color palette
Good spacing and typography
Smooth transitions and hover effects
Subtle animations (not overwhelming)

⚙️ Technical Details
Built with HTML, CSS, and JavaScript
Fully responsive (mobile-first approach)
Clean and structured code
Optimized for fast loading

🚀 Focus
Strong UI/UX design
Smooth and intuitive user interactions
Simple, elegant, and professional presentation
